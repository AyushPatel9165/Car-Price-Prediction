 # Car Price Prediction

## Overview
This is a Machine Learning project that predicts car prices based on various features such as car brand, year, fuel type, and kilometers driven. The project includes data preprocessing, model training, and deployment using Flask.

## Problem Statement
The objective is to predict the selling price of a car using Machine Learning regression algorithms.

## Features
- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Model training using regression algorithm
- Model evaluation
- Flask web application for prediction

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Flask
- HTML/CSS

## Project Structure

## How to Run

Step 1: Install dependencies
pip install -r requirements.txt

Step 2: Run the application
python app.py

## Output
The system will predict car price based on user input.
